# 🧮 Simple Calculator in Python

This is a simple **menu-driven calculator** built using Python.
It performs basic operations using basic logic instead of built-in operators for multiplication and division.

---

## 🚀 Features

* Addition
* Subtraction
* Multiplication (using repeated addition)
* Division (using repeated subtraction)
* Modulo (remainder calculation)
* Menu-driven user input

---

## 🛠️ How It Works

This program uses a `Calculator` class that contains different methods:

* `add()` → adds two numbers
* `subtract()` → subtracts two numbers
* `multiply()` → adds a number repeatedly
* `divide()` → subtracts repeatedly to find quotient
* `modulo()` → finds remainder using subtraction

---

## ▶️ How to Run

1. Make sure Python is installed
2. Clone this repository or download the file
3. Run the program:

```bash
python main.py
```
